# Class 7 Guide & Textbook Library - All Books Dataset

এই রিপোজিটরিতে সপ্তম শ্রেণির সকল বিষয়ের পাঠ্যবই এবং গাইড বইয়ের একটি সুসংগঠিত পিডিএফ (PDF) সংগ্রহ রয়েছে। এটি মূলত একটি অ্যান্ড্রয়েড অ্যাপ্লিকেশনের ব্যাকএন্ড ডেটাবেস এবং ক্লাউড স্টোরেজ হিসেবে ব্যবহৃত হচ্ছে।

## 📱 অ্যাপের বৈশিষ্ট্য (App Features)
* **সকল বিষয়:** বাংলা, ইংরেজি, গণিতসহ সকল বিষয়ের গাইড ও নোট।
* **বিভাগভিত্তিক কন্টেন্ট:** গদ্য, পদ্য, আনন্দপাঠ এবং টেক্সটবুক আলাদা ট্যাবে বিন্যস্ত।
* **ডাউনলোড সিস্টেম:** চ্যাপ্টারভিত্তিক ডাউনলোড করে অফলাইনে পড়ার সুবিধা।
* **ইউজার ইন্টারফেস:** ডার্ক মোড এবং ফেভারিট লিস্ট সুবিধা সংবলিত আধুনিক UI।

## 🛠 টেকনিক্যাল আর্কিটেকচার (Technical Architecture)
* **Frontend:** অ্যান্ড্রয়েড অ্যাপ্লিকেশন (Jetpack Compose)।
* **Data Source:** `data.json` ফাইলের মাধ্যমে ডাইনামিক ডেটা লোড।
* **Storage:** GitHub Releases ব্যবহার করে হাই-স্পিড পিডিএফ হোস্টিং।
* **Direct Links:** নিরবচ্ছিন্ন পড়ার জন্য সরাসরি ডাউনলোড লিঙ্ক ইন্টিগ্রেশন।

## 📂 ফাইল অর্গানাইজেশন (Organization)
এই প্রজেক্টের সকল পিডিএফ ফাইল GitHub Releases-এ বিভিন্ন ট্যাগের অধীনে আপলোড করা হয়েছে:
- `ban1st`, `ban2nd`, `math`, `science` ইত্যাদি আলাদা রিলিজ ট্যাগের মাধ্যমে ফাইলগুলো ম্যানেজ করা হয়।
- মোট ডেটা সাইজ: প্রায় ৩ জিবি।

## 🚀 ব্যবহারবিধি (Usage)
ডেভেলপাররা চাইলে এই `data.json` ফাইলটি ব্যবহার করে কন্টেন্ট স্ট্রাকচার সম্পর্কে ধারণা নিতে পারেন। অ্যাপের প্রতিটি ফাইল নিচের ফরম্যাটে এক্সেস করা হয়:
`https://github.com/appbd/Class7Books/releases/download/{tag}/{filename}.pdf`
